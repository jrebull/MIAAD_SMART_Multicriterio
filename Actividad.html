Práctica de Laboratorio: Análisis de Estrategias Multicriterio

Materia: Optimización Inteligente — Maestría

1. Objetivo de la Práctica

El objetivo es comprender cómo el momento en que el tomador de decisiones expresa sus preferencias (A priori vs. A posteriori) altera el resultado final. Analizaremos un problema con dos objetivos en conflicto: Minimizar Costo Operativo (f1) y Minimizar Tiempo de Entrega (f2).

Preparación del entorno: Utilizaremos la librería pymoo. Instálala en tu entorno virtual o en Google Colab usando:
pip install pymoo matplotlib numpy

2. Definición del Problema

Comenzaremos definiendo nuestro problema base. Copia este código en tu script o celda de Jupyter:

import numpy as np
import matplotlib.pyplot as plt
from pymoo.core.problem import ElementwiseProblem

# Definimos un problema continuo simulando Costo vs Tiempo
class RoutingProxyProblem(ElementwiseProblem):
    def __init__(self):
        super().__init__(n_var=2, n_obj=2, xl=np.array([0.0, 0.0]), xu=np.array([5.0, 5.0]))

    def _evaluate(self, x, out, *args, **kwargs):
        f1 = 4 * x[0]**2 + 4 * x[1]**2              # f1: Costo Operativo
        f2 = (x[0] - 5)**2 + (x[1] - 5)**2          # f2: Tiempo de Entrega
        out["F"] = [f1, f2]

problem = RoutingProxyProblem()

3. Estrategia 1: Enfoque A Posteriori (NSGA-II)

Primero, dejaremos que el algoritmo NSGA-II aproxime todo el Frente de Pareto. Una vez obtenido el conjunto de soluciones no dominadas, simularemos la intervención del tomador de decisiones.

from pymoo.algorithms.moo.nsga2 import NSGA2
from pymoo.optimize import minimize

# 3.1 Ejecutar NSGA-II
algorithm_nsga2 = NSGA2(pop_size=100)
res_nsga2 = minimize(problem, algorithm_nsga2, ('n_gen', 100), seed=1, verbose=False)

F_pareto = res_nsga2.F  # Valores de las funciones objetivo en el Frente

# --- POST-PROCESAMIENTO: EL TOMADOR DE DECISIONES ELIGE ---

# A) Elección por Suma Ponderada sobre el Frente (Post-selección)
# Normalizamos los valores del frente para evaluarlos equitativamente
F_norm = (F_pareto - F_pareto.min(axis=0)) / (F_pareto.max(axis=0) - F_pareto.min(axis=0))

pesos = np.array([0.7, 0.3]) # 70% importancia al Costo, 30% al Tiempo
valores_ponderados = np.sum(F_norm * pesos, axis=1)
idx_mejor_ponderado = np.argmin(valores_ponderados)
solucion_post_ponderada = F_pareto[idx_mejor_ponderado]

# B) Elección Lexicográfica sobre el Frente (Post-selección)
# Prioridad 1: Tiempo de entrega (f2). Prioridad 2: Costo (f1).
# Ordenamos el frente basándonos en f2.
idx_lexicografico = np.argsort(F_pareto[:, 1])[0] # Tomamos el índice del menor f2
solucion_post_lexicografica = F_pareto[idx_lexicografico]

print(f"Solución NSGA-II + Suma Ponderada: Costo={solucion_post_ponderada[0]:.2f}, Tiempo={solucion_post_ponderada[1]:.2f}")
print(f"Solución NSGA-II + Lexicográfica: Costo={solucion_post_lexicografica[0]:.2f}, Tiempo={solucion_post_lexicografica[1]:.2f}")

4. Estrategia 2: Enfoque A Priori (Monoobjetivo Directo)

Ahora, asumiremos que no tenemos tiempo de calcular el frente de Pareto completo. El tomador de decisiones inyecta sus preferencias matemáticamente antes de ejecutar la optimización, convirtiendo el problema en monoobjetivo y resolviéndolo con un Algoritmo Genético Clásico (GA).

from pymoo.algorithms.soo.nonconvex.ga import GA

# A) Suma Ponderada Directa (A Priori)
class WeightedSumProblem(ElementwiseProblem):
    def __init__(self):
        super().__init__(n_var=2, n_obj=1, xl=np.array([0.0, 0.0]), xu=np.array([5.0, 5.0]))
    def _evaluate(self, x, out, *args, **kwargs):
        f1 = 4 * x[0]**2 + 4 * x[1]**2
        f2 = (x[0] - 5)**2 + (x[1] - 5)**2
        # Escalamiento manual simplificado y aplicación de pesos (0.7, 0.3)
        out["F"] = 0.7 * (f1 / 200.0) + 0.3 * (f2 / 50.0)

res_apriori_pond = minimize(WeightedSumProblem(), GA(pop_size=100), ('n_gen', 100), seed=1)
# Evaluamos la solución encontrada en las funciones originales para comparar
x_pond = res_apriori_pond.X
f1_pond = 4 * x_pond[0]**2 + 4 * x_pond[1]**2
f2_pond = (x_pond[0] - 5)**2 + (x_pond[1] - 5)**2


# B) Lexicográfica Directa (A Priori)
# Solo optimizamos el objetivo más importante: Tiempo (f2)
class LexicographicProblem(ElementwiseProblem):
    def __init__(self):
        super().__init__(n_var=2, n_obj=1, xl=np.array([0.0, 0.0]), xu=np.array([5.0, 5.0]))
    def _evaluate(self, x, out, *args, **kwargs):
        f2 = (x[0] - 5)**2 + (x[1] - 5)**2
        out["F"] = f2

res_apriori_lex = minimize(LexicographicProblem(), GA(pop_size=100), ('n_gen', 100), seed=1)
x_lex = res_apriori_lex.X
f1_lex = 4 * x_lex[0]**2 + 4 * x_lex[1]**2
f2_lex = (x_lex[0] - 5)**2 + (x_lex[1] - 5)**2

print(f"Solución GA Ponderado (A Priori): Costo={f1_pond:.2f}, Tiempo={f2_pond:.2f}")
print(f"Solución GA Lexicográfico (A Priori): Costo={f1_lex:.2f}, Tiempo={f2_lex:.2f}")
